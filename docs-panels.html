Before publishing

Required foundation

  1. Run /install wizard.
  2. Verify the game profile and edition.
  3. Configure guild and world with /guild set.
  4. Map ranks, Guest and global moderator roles.
  5. Place Luminox above every role it must manage.
  6. Publish each panel in a dedicated channel.
