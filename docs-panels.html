Before publishing

Do setup in the right order

Panels are the public interface of Luminox. Before publishing them, configure the guild, world and roles so the panels can validate users correctly.

  1. Run /set setup to see what is missing.
  2. Configure guild and world with /set guild.
  3. Configure moderator roles with /set moderators role.
  4. Map guild ranks and guest role with /set ranks.
  5. Configure special roles such as GuildBank treasurer and Guards alert role.
  6. Publish each panel in its dedicated channel.

Panel setup list

Commands for every control panel

Registration/registration setup channel:#channel. Publishes Register character, My characters and Change main character.
Loot Split/loot-panel channel:#channel. Publishes Split loot and How it works.
GuildBank/set guildbank log channel:#channel. Then use /set guildbank treasurer-role and optional /set guildbank minimum.
Leaderboards/set leaderboards channel:#channel. Publishes index and leaderboard messages.
Guild List/set guildlist channel:#channel. Publishes the full guild member list.
Levels/set levels list channel:#channel. Publishes level-up messages and daily/monthly gains panel.
Deaths/set deaths list channel:#channel. Publishes death messages and feeds death boards.
Finder/set finder channel:#channel. Publishes matches, looking-for-team and history controls.
Guards/set guards channel:#channel role:@Alerts. The alert role is required.
Support/support setup channel:#channel. Then add categories with /support category.
Hunts/set hunts channel:#channel. Publishes Hunt Board.
Bosses/set bosses channel:#channel. Publishes Boss Board.
Quests/set quests channel:#channel. Publishes Quest Board.
Events/set events channel:#channel. Publishes general Event Board.
Watchlists/set watchlist type:... channel:#channel. Assigns member/enemy watchlist channels.
Blacklist/set blacklist channel:#channel. Publishes blacklist panel.
Ban List/set banlist channel:#channel. Publishes ban list panel.
Tracker/set tracker channel:#channel. Publishes risk character change log.
Loyalty/set loyalty channel:#channel. Publishes loyalty profile panel.
Message Loyalty/set loyalty-messages channel:#channel enabled:true. Enables chat activity rewards in selected channels.
Moderation Log/set log channel:#channel. Sends moderation-style bot logs.
Autodelete/set autodelete channel:#channel minutes:number. Cleans messages after the configured time.

Refresh

How panels update

Most panels edit existing messages instead of posting new ones. Use /sync all to refresh visible panels. Use /update-all only when you need the heavier registered character sync.

Permissions

If a panel does not appear

Check that Luminox can view the channel, send messages, embed links and manage messages where cleanup is expected. Event boards also need reactions and thread permissions.
