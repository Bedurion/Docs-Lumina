Documentation · Commands

Commands and panel setup, grouped by access level.

Luminox is panel-first. Members should normally click panels; admins use commands to publish and configure those panels.

Panel setup commands

/registration setup Admin

Publishes the character registration panel in the selected channel.

/loot-panel Admin

Publishes the Loot Split panel with Split loot and How it works buttons.

/set guildbank log Admin

Assigns the GuildBank channel and publishes the bank control panel.

/set hunts, /set bosses, /set quests, /set events Admin

Assign event board channels and publish their Create and Information panels.

/support setup Admin

Publishes the private support ticket panel. Add categories with /support category.

/set guards Admin

Publishes Guards and requires the alert role used for notifications.

/set finder, /set leaderboards, /set guildlist Admin

Assign Finder, Leaderboards and Guild List panel channels.
