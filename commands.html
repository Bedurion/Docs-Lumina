Documentation · Commands

Every active command, with its real access level.

This reference follows the 31 commands currently registered by Luminox. Member actions such as registration, loot splitting, event creation and list management happen through buttons rather than duplicate commands.

Visibility and execution

Commands marked Administrator normally use Discord's Administrator default permission. /finder and /loyalty can remain visible because their modules also power public panel interactions, but their setup subcommands still reject non-administrators. Panel buttons can separately authorize configured moderators or specialist roles.

Everyone

/infoEveryone

Shows the caller's registrations, main character, Loyalty data and other account information stored by Luminox.

/updateEveryone

Refreshes your own main character, Discord nickname and managed guild role.

/timezone set zone:Europe/MadridEveryone

Sets the personal timezone used while entering event dates and viewing personalized time information.

/whois character:NameEveryone

Shows the registered Discord owner, account characters, Loyalty, GuildBank contribution and Tibia details for a verified guild character.

Apps → Convert timeEveryone

Message context command that converts a detected time from the author's timezone to the caller's timezone.

Installation, identity and access

/install wizardAdministrator

Non-destructive setup wizard. Detects missing configuration and links to the correct commands.

/install overviewAdministrator

Shows the complete current server profile, channels, roles, panel configuration and command access overview.

/install refreshAdministrator

Refreshes configured panels without running the heavy registered-character Tibia scan.

/guild set · /guild chatAdministrator

Registers or removes the verified Tibia guild/world and assigns the Guild Chat channel for important alerts.

/ranks map · /ranks guest · /ranks overviewAdministrator

Maps Tibia guild ranks to Discord roles, configures Guest and inspects the current mapping.

/moderators roles add|remove|overviewAdministrator

Maintains multiple global moderator roles used across support, moderation and specialist systems.

/transfer accountAdministrator

Transfers all Luminox registrations and account-linked data from one Discord account to another.

Panel and channel configuration

/registration set · /registration refreshAdministrator

Publishes or refreshes the Registration panel containing Register, My characters, Change main, Delete and Timezone controls.

/guildlist setAdministrator

Assigns the automatic full guild member list channel.

/leaderboards setAdministrator

Assigns the combined leaderboard channel and publishes its clickable index.

/levels set · /deaths setAdministrator

Assigns level and death announcement channels and their summary panels.

/watchlist set · /watchlist overviewAdministrator

Assigns Guild Members Online or Enemies Online and inspects their configuration.

/finder setAdministrator

Publishes the Finder panel with matches, Looking for team and Match history.

/guards setAdministrator

Publishes Guards and assigns the role used for guard activation and enemy-report notifications.

/events set hunts|bosses|quests|generalAdministrator

Assigns each event board channel. Event creation itself happens through the board's Create button.

/loot setAdministrator

Publishes the Loot Split panel. The old member-facing loot command is intentionally retired.

/autodelete setAdministrator

Configures message cleanup in one channel from 10 minutes to 30 days.

/moderation setAdministrator

Assigns the audit channel for registrations, role changes and moderation-list actions.

/moderators set voting|applicationsAdministrator

Publishes the private staff voting panel or public staff application panel.

Economy, Loyalty and recruitment

/guildbank set log|treasurer|minimumAdministrator

Assigns the control panel/log channel, specialist review role and deposit/review thresholds.

/guildbank overview · /guildbank refreshAdministrator

Inspects GuildBank moderation status or moves a refreshed control panel to the bottom of the channel.

/loyalty setAdministrator

Assigns either the public Loyalty activity panel or the Server Boost log to the selected channel.

/loyalty messages · /loyalty streamsAdministrator

Enables or disables message rewards in one text channel or Stream Loyalty in one voice channel.

/loyalty-adjustAdministrator

Adds or subtracts account Loyalty Points with a mandatory logged reason.

/recruitment setAdministrator

Publishes Recruitment Rewards and assigns the additional specialist review role.

Moderation and support

/blacklist set · /blacklist overviewAdministrator

Publishes or inspects Blacklist. Add, edit and delete actions are panel buttons.

/banlist set · /banlist overviewAdministrator

Publishes or inspects Ban List. Record management is panel-first.

/tracker setAdministrator

Assigns the permanent identity-change log and live Tracker panel channel.

/support setAdministrator

Publishes the private Support panel in a clean text channel.

/support categoryAdministrator

Creates a category with description, up to five specialist staff roles and an optional minimum user role.

/support edit-category · /support remove-categoryAdministrator

Fully edits or removes an existing category through autocomplete selection.

/support overview · /support refreshAdministrator

Shows private configuration or redraws the Support panel.

/update-allAdministrator

Runs the full registered-character synchronization for the current server. Use only when a heavy refresh is actually required.

Panel handlers are not slash commands

/register, member-facing /loot, event creation commands and Blacklist/Ban List CRUD commands are replaced by their channel panels. Internal modules can still route buttons, selectors and modals, but Discord does not register them as member-facing commands.
