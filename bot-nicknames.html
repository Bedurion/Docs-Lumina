Choose the identity source

Simple when it should be. Verified when it can be.

The selected edition determines where the game identity comes from. The visible result remains compact, predictable and connected to the correct Discord account.

Universal Edition

One member-provided game name

The member enters the name they use in their game. Luminox stores it only for that Discord server and displays Game Name [L#].

Community / Founder

One verified Tibia main

The member verifies character ownership. Luminox displays Name [VOCLevel L#] and follows the selected main.

Automatic Loyalty suffix

The L value belongs to the Discord account. When a reward or reversal changes its Loyalty Level, the nickname refreshes automatically.

Server-safe uniqueness

Universal game names are unique inside that server. Community identities inherit uniqueness from verified Tibia ownership.

Panel-first changes

Members use guided buttons to set a game name, review their profile, choose a Tibia main where supported or remove the stored profile.

Honest permission handling

If Discord hierarchy blocks an edit, Luminox saves the profile and reports the exact suggested nickname instead of claiming success.
