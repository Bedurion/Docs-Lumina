Bot · Member identity

Verify who controls each Tibia character.

Luminox is a Discord bot that verifies character ownership through a temporary Tibia.com comment code. The verified account can then power mains, nicknames, guild roles and every connected Community feature.

See the registration stepsView Community plans

In plain English

One Discord account becomes one trusted guild profile.

A member enters an exact character name. Luminox gives them a temporary verification code to place in that character's public Tibia.com comment. When the code is found, the bot knows the person controls the character.

The member can register more characters, choose a main and see everything from one panel. Staff no longer need to compare screenshots or manually maintain every nickname.

Who does what?

Simple for members. Controlled by administrators.

Member experience

A guided panel, not a command lesson.

Members click Register character, follow the code instructions, select a timezone when needed and later manage their characters or change their main from the same channel.

Administrator setup

Define the identity boundary once.

Administrators verify the guild, configure its world, map guild ranks to Discord roles and publish the Registration panel in the chosen channel.

What happens automatically?

The verified profile powers the server.

World protection

A character from a different world cannot silently enter a server configured for Secura.

Main identity

The chosen main represents the account in public guild panels and linked workflows.

Role synchronization

Eligible guild ranks can map to Discord roles without daily manual checks.

Readable nicknames

Character, vocation, level and Loyalty level stay visible in one compact format.
