Before you contact us

Include enough context for a useful answer.

  • The Discord server and channel where the issue occurred.
  • The exact panel, button or registered slash command involved.
  • The approximate time and any visible Discord error.
  • Whether the caller is a member, moderator, specialist or administrator.
  • A screenshot without exposing tokens, secrets or private member evidence.

Security

Never share the bot token

Luminox staff should never need your Discord bot token, environment file or raw private database to answer a normal setup question. Remove sensitive identifiers before posting publicly and use a private Support ticket for evidence that should not be visible to the community.
