Channel cleanup

Delete ordinary messages by age

/autodelete set channel:#utility minutes:60

Configure one channel from 10 minutes to 30 days. Running the command again updates that channel's policy. Managed panels and system messages use their own lifecycle rules.

Stable panels

Reuse when possible, repost only when position matters

Most panels edit their existing Discord message. Final action panels such as GuildBank, Loot or Event Board may move to the bottom when new workflow content is published. Restarting the bot should not create duplicate stable panels.

Private responses

Expiry follows usefulness

Very shortSuccess acknowledgements and errors that no longer require action.
Reading timeOverviews, paginated histories and configuration details.
RegistrationVerification-code responses remain long enough to update Tibia.com and are removed when the workflow completes.

Discord controls the underlying interaction token lifetime; Luminox schedules deletion within that supported window.

Local backups

Automatic, rotating and owner-operated

  • One automatic backup every 24 hours.
  • Keep the latest 30 daily backups.
  • Create one monthly backup on the first day.
  • Keep 12 monthly backups.
  • Backup commands are not exposed to administrators of every installed Discord server.

Sustainable data

Balances outlive detailed history

Luminox keeps authoritative balances and necessary audit links while pruning old reward detail, reviewed transaction detail and temporary state. Discord-native threads carry full Support and Event conversation history.
