Documentation · Installation

Install safely without erasing an existing server configuration.

The setup wizard audits what is missing. It does not reset channels, roles, registrations or historical data.

Step 1

Add Luminox to Discord

Invite Luminox with its bot and application-command scopes, choose the target server and place its managed role above every guild, Guest, moderator and specialist role it must assign or remove.

Add Luminox to your Discord

Step 2

Grant only the permissions used by configured systems

PermissionWhy it is needed
View Channels / Read HistoryRead and refresh configured panels and threads.
Send Messages / Embed LinksPublish panels, logs, announcements and private workflow responses.
Manage MessagesClean guarded or autodelete channels and reposition final control panels.
Manage ThreadsCreate, lock and archive event/support/report threads.
Manage RolesSynchronize guild ranks, Guest and opt-in alert roles.
Manage NicknamesMaintain Name [RP360 L8]. Discord never allows bots to change the server owner's nickname.

Step 3

Run the non-destructive wizard

/install wizard

The wizard shows the selected edition/profile, completed configuration and missing items with the exact command required. Run it again at any time; existing values remain untouched unless you explicitly replace them.

Step 4

Verify the Tibia guild and world

/guild set name:GuildName world:Secura

Luminox provides a verification code that must be placed in the guild description. The verified world becomes the registration boundary: characters from other worlds cannot register on that Discord server.

Step 5

Map access before publishing member panels

  1. Add every global moderator role with /moderators roles add.
  2. Map each Tibia rank using /ranks map.
  3. Configure the Guest role with /ranks guest.
  4. Add Treasurer, Recruitment, Guildhall and Support specialist roles where required.
  5. Use the relevant overview commands to verify the mapping.

Step 6

Publish panels in dedicated channels

Start with Registration, Guild List, Leaderboards, Members Online and Support. Add economy, events, intelligence and staff panels afterward. The complete checklist is available in the Panel Directory.

Final verification

Inspect before inviting members

/install overview
  • Confirm profile and edition.
  • Confirm verified guild/world.
  • Confirm global moderator and specialist roles.
  • Open each linked configured channel.
  • Test Registration with a real character.
  • Use /install refresh if panels need a light redraw.
