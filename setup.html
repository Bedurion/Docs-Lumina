Setup · Install

Install, run and configure Luminox.

This page is for bot owners and server admins.

Install

npm install

Environment

cp .env.example .env

Start

npm start

Development

npm run dev

macOS launchers

  1. setup.command prepares setup.
  2. start.command starts normally.
  3. dev.command restarts on code changes.
  4. check.command validates token setup.

Correct shutdown

Use Control + C in the terminal window.

After install

Configure panels before inviting everyone

Once the bot is online, use /set setup in Discord to see what is missing. Then publish the public control panels: registration, loot, GuildBank, events, Finder, Guards, support, leaderboards and watchlists.

Open panel setup guide

Important

Do not rely only on commands for members. Luminox is designed around guided panels so users do not need to memorize syntax.
