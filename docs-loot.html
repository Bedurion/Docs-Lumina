Documentation · Loot

Calculate party hunt transfers with GuildBank rules.

The loot system parses Tibia session logs, calculates equal split transfers and applies optional GuildBank contributions.

Panel setup

How to configure the Loot Split panel

Admins publish the loot control panel with /loot-panel channel:#loot. Members should use this button flow instead of typing the legacy loot command.

Command/loot-panel channel:#channel
PublishesSplit loot and How it works buttons.
Optional setupConfigure GuildBank log and minimums if loot contributions should be tracked.
RefreshUse /sync all if the panel is missing or not at the bottom.

Basic use

Paste full session log

  1. Run /loot.
  2. Paste complete Tibia session data.
  3. Choose Market or Leader mode.
  4. Choose GuildBank contribution.
  5. Review transfers.

Modes

Market and Leader

MarketOptional looter compensation.
LeaderLeader-style loot handling.
No looterNormal split without looter compensation.

GuildBank

Minimum and loyalty

  • Minimum tracked contribution is 1kk.
  • Only registered accounts receive loyalty credit.
  • Transfers are logged for review.

Confirmation

Audit trail

Transfer confirmation updates the GuildBank log from pending to confirmed and links back to the loot session.
