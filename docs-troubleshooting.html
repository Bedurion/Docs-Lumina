Registration

Character cannot verify

  • Check world.
  • Put code at start of comment.
  • Wait for Tibia.com update.
  • Try same button again.

Roles

Hierarchy matters

  • Luminox role must be high enough.
  • Bots cannot change server owner nickname.
  • Unregistered users should not keep guild rank roles after cleanup.

Panels

When stale

  • Use /sync all for visible panels.
  • Use /update-all for registered character data.
  • Check Embed Links and Manage Messages permissions.

Discord limitations

Client cache

Some Discord sidebar and cache behavior cannot be forced instantly by bots.
