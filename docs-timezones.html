Documentation · Timezones

International guilds need clear time.

Luminox stores user timezones and uses Discord timestamps.

User setup

How users configure timezone

Users can configure timezone with /set timezone zone:.... During first character registration, Luminox can also ask for timezone automatically so the user does not need to find the command manually.

User command/set timezone zone:Europe/Madrid or another valid timezone.
Registration flowIf timezone is missing, the registration panel guides the user through it.
Used byEvent scheduling, friendly date parsing and time conversion.

Set timezone

When required

Timezone is required during first registration so event creation can interpret local time safely.

Events

Scheduling

  • Creators enter times in their own timezone.
  • Discord renders timestamps per viewer.
  • Recruiting mode works when date is unknown.

Friend Time

React to a time
