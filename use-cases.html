For growing guilds

Replace repeated staff work before adding more staff.

When a guild grows, the same questions repeat: Who is this member? Which rank should they have? Who approved this contribution? Is this event still active? Luminox turns those answers into shared, current panels.

  • Members complete guided actions without memorizing commands.
  • Moderators receive access through global and specialist roles.
  • Administrators can audit configuration without resetting it.
  • Permanent history stays readable in Discord instead of hidden in one operator's files.

Recommended first stack

  1. RegistrationEstablish trusted account identity.
  2. Role mappingRemove manual access drift.
  3. SupportCreate one private help route.
  4. EventsMove participation into boards.
