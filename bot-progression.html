In plain English

The bot notices meaningful character changes.

When a registered character gains a level, Luminox publishes one linked announcement and updates the rolling level panel. Deaths use the same principle: only genuine new records become announcements.

The live character record is the source of truth. A manually edited Discord nickname never overrides the current Tibia level shown in panels, lists or announcements.
