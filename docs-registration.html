Documentation · Registration

Verified identity is the foundation of Luminox.

Registration links Discord accounts to Tibia characters and powers nicknames, roles, leaderboards and events.

Panel setup

How to configure the Registration panel

Admins publish the registration control panel with /registration setup channel:#registration. This panel should be placed in the channel where new members verify their characters.

Command/registration setup channel:#channel
PublishesRegister character, My characters and Change main character buttons.
Required beforeConfigure guild and world with /set guild so Luminox knows which world is allowed.
RefreshUse /registration refresh or /sync all if the panel needs to be redrawn.

Registration flow

How verification works

  1. Click Register character.
  2. Enter exact Tibia character name.
  3. Set timezone if this is your first registration.
  4. Paste the verification code at the start of your Tibia character comment.
  5. Click Register character again with the same name.

World and guild rules

What can be registered

  • The character must be on the configured world.
  • For Lumina this means Secura.
  • The character does not have to be in Lumina to be registered.
  • A Discord account can register up to 10 characters.

Main character

What it controls

NicknameCharacter name plus vocation and level.
RoleGuild rank role if main belongs to the configured guild.
LeaderboardsAccount totals display through current main.

Common problems

Why verification fails

  • Code is not at the start of the comment.
  • Tibia.com has not updated yet.
  • Character is on another world.
  • Bot lacks nickname or role permissions.
