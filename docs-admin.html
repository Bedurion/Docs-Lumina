Installation

Wizard plus permanent overview

/install wizard/install overview

The wizard detects missing configuration and offers only the next safe steps. It does not erase existing channels, guild verification, roles or data. Overview always shows the active profile and current configuration.

Discord permissions

Administrator is a Discord permission

Luminox does not maintain a second administrator list. A Discord member with the Administrator permission receives administrator access. Place the bot role above every rank, Guest, Alerts and specialist role it must manage.

  • View Channels and Send Messages.
  • Embed Links and Read Message History.
  • Manage Messages and Manage Threads where panels operate.
  • Manage Channels for automatic counters.
  • Manage Roles and Manage Nicknames for identity sync.
  • Add Reactions for event compatibility.

Staff access

Multiple global moderator roles

/moderators roles add role:@Vice Leader/moderators roles overview

Specialist actions add a second requirement: Treasurer for GuildBank review, Recruitment for recruitment review and category roles for Support. Administrators bypass specialist-role requirements.

Refresh tools

Choose the lightest correct operation

/install refreshRefreshes configured panels and avoids heavy Tibia character scans.
/registration refreshReposts only the Registration panel.
/guildbank refreshMoves the GuildBank control panel to the bottom.
/update-allAdministrator-only registered-character refresh with cooldown.
/updateMember refresh for personal registered data.

Retention

Bounded metadata and Discord-native archives

  • Automatic local backups run every 24 hours, retain 30 daily copies and 12 monthly copies.
  • Support and event conversations remain Discord threads rather than duplicated transcripts.
  • Loyalty reward detail, GuildBank review data and temporary states have retention rules.
  • /autodelete set manages utility-channel message age from ten minutes to thirty days.

Operations

Stop cleanly with Control-C

Run the bot with the project start or development script and stop the active process with Control + C. Backup management commands are intentionally not published to every Discord server.
