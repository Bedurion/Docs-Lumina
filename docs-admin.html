Setup · Admins

Configure once, then let Luminox keep itself clean.

Admin tools focus on safe configuration, role mapping, channel assignment and manual correction.

Recommended order

Start with overview

  1. Run /set setup.
  2. Select profile if missing.
  3. Configure guild and world.
  4. Configure roles and ranks.
  5. Publish panels.

Role hierarchy

Discord permissions

  • Luminox role must be high enough.
  • Treasurer can be separate from moderator.
  • Support categories can have separate roles.

For the exact command that publishes each panel, use the dedicated control panel setup guide.

Sync tools

Use right refresh

/sync allRefresh visible panels.
/update-allRefresh registered characters.

Safety

Backups and cleanup

  • Automatic backups.
  • Autodelete for utility channels.
  • Moderation logs for important changes.
