Private support without messy channels.

Support tickets create private Discord threads for users and responsible staff roles.

Panel setup

How to configure the Support panel

Admins publish the support panel with /support setup channel:#help. After that, at least one support category must be created before users can open tickets.

Panel command/support setup channel:#channel
Add category/support category creates a selectable ticket reason.
Edit category/support edit-category changes name, description, staff roles and minimum role.
Refresh/support refresh redraws the panel.

Opening a ticket

Member flow

  1. Open support panel.
  2. Choose category.
  3. Fill requested questions.
  4. Continue inside private thread.

Staff workflow

Claim and resolve

  • Staff can claim, unclaim and transfer.
  • User can mark resolved.
  • Staff can close ticket.

Closed tickets

Where they go

Closed tickets are locked and archived Discord threads, findable through archived threads or My Tickets.

Categories

Permissions

Categories can define minimum user roles and staff roles.
