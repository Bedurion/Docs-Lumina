Channels

Three independent destinations

Levels/levels set channel:#levels
Deaths/deaths set channel:#deaths
Guild List/guildlist set channel:#guild-list

Level announcements

One compact message with rolling context

Verified gains show the linked character, vocation, current level and totals for the last 24 hours, 30 days and 365 days. Daily is a rolling 24-hour period, not midnight-to-midnight.

Example

Advanced to level 362. Now has a total of 3 daily levels, 11 monthly and 11 yearly.

Unverified downward polling changes are not announced as real level loss. This prevents false level-down messages caused by stale or inconsistent external snapshots.

Death announcements

Each real Tibia death is counted once

Luminox uses the official death entries shown on the Tibia character page instead of guessing from level changes. The announcement links the registered Discord account when available, bolds the killer and includes rolling monthly and yearly death totals.

Rolling panels

The summary always returns to the bottom

  • Top ten appears in the first message.
  • All remaining qualifying characters continue in additional messages.
  • The final message links back to the Top 10.
  • New announcements trigger panel repositioning so the summary stays after individual messages.
  • Monthly means 30 days; yearly means 365 days.

Guild List

Ranks, registration and power

  • Members are grouped under Tibia guild-rank headings.
  • Rank headings repeat when a group continues in another message.
  • Every continuation message includes Go to top.
  • Registered characters use Discord-account links and compact vocation, level and Loyalty notation.
  • Unregistered status remains visible without repeating unnecessary Discord usernames.
  • Guild Power is the sum of current member levels.

Data integrity

Reuse fresher character snapshots

Guild List and progression panels reuse fresher registered-character and online observations where available instead of waiting for a character to disconnect. Deduplication keys stop one historical death or level event from being emitted repeatedly.
