One member profile

Verify once. Reuse the result.

A verified main character can set the correct nickname and role, identify the member in events, connect GuildBank rewards and represent the same person in Leaderboards, Support and Recruitment.

  • Members do not create a separate profile for every feature.
  • Current information is reused instead of requested again.
  • Buttons replace public commands when a guided form is easier.
  • Ticket conversations and attachments remain stored by Discord.
