Connected workflow

Information entered once becomes useful everywhere.

A verified main character can determine a member's nickname and role, identify their event character, attribute GuildBank rewards, represent them in Leaderboards and make their support, recruitment and staff activity auditable.

  • No duplicate member profiles across modules.
  • No extra Tibia request when information already collected answers the question.
  • No public command clutter for actions that work better as guided panels.
  • No second transcript database for conversations already stored by Discord.
