Registration

Your Discord access follows the verified character.

The Registration panel asks for a character name and gives you a temporary verification code. Place that code at the beginning of the character comment on Tibia.com, then use Register character again with the same name.

  • Only characters from the configured world can register.
  • A guild member receives the Discord role mapped to their Tibia guild rank.
  • A registered character outside Lumina can receive Guest access when configured.
  • Timezone is included in the first registration flow and remains saved if verification needs another attempt.
