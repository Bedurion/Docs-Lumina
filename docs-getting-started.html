For members

First steps

  1. Open the registration panel.
  2. Set timezone when asked.
  3. Register and verify your Tibia character.
  4. Use panels for events, finder, guards, support and loyalty.

For admins

Setup order

  1. Run /set setup.
  2. Configure guild, ranks and moderators.
  3. Publish registration, leaderboards, watchlists and support panels.
  4. Use sync tools when panels need refreshing.
