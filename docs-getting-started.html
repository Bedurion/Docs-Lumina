Members

Your first five minutes

Open Registration

Click Register character in the configured registration channel.

Set timezone

The first registration flow asks automatically if no timezone exists.

Verify character

Place the temporary code at the beginning of the Tibia comment and submit again.

Use panels

Join events, find teams, split loot, open support and inspect Loyalty.

Use /update if your own character, nickname or role needs an immediate targeted refresh.

Staff

Know the difference between global and specialist access

  • A configured global moderator role enables common moderation workflows.
  • GuildBank review additionally requires Treasurer.
  • Recruitment review additionally requires the configured Recruitment role.
  • Support categories can require one of several category-specific staff roles.
  • Discord administrators retain administrative access and every action remains attributable.

Administrators

Use a safe configuration order

  1. /install wizard
  2. /guild set
  3. /moderators roles add
  4. /ranks map and /ranks guest
  5. Publish panels from the Panel Directory
  6. /install overview
Never rebuild a working server unnecessarily

The wizard detects missing configuration. Re-running it does not require deleting existing channels, panels or registrations.

Daily operation

Most systems maintain themselves

Every minute

Founder online checks, Watchlists, Finder and Guards reuse information already collected by Luminox.

Every few minutes

Event and Loyalty panels refresh automatically.

Every ten minutes

Leaderboards, Guild List, Blacklist and registered characters complete their normal cycles.

Use manual refresh commands only when the automatic interval is not appropriate for the situation.

Help

Use the private Support panel

Open the configured Support channel, choose the correct category and answer every required question. The resulting thread is private to you and responsible staff.
