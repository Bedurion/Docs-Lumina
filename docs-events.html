Panel setup

How to configure event boards

Each event type has its own channel and board panel. The board publishes Create and Information buttons, shows active events and stays near the bottom of the channel.

Hunts/set hunts channel:#hunts
Bosses/set bosses channel:#bosses
Quests/set quests channel:#quests
Events/set events channel:#events
RefreshUse /sync all if a board or channel counter needs to be redrawn.
